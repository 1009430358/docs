=================================
Contribute to the MongoDB Project
=================================

.. default-domain:: mongodb

.. contents:: On this page
   :local:
   :backlinks: none
   :depth: 1
   :class: singlecol

There are `many ways to contribute
<http://www.mongodb.org/get-involved>`_ to the MongoDB project. For
instance, you can:

#. Answer MongoDB questions on `StackOverflow <http://stackoverflow.com/questions/tagged/mongodb?sort=newest&pageSize=15>`_ or the `MongoDB-user Google Group <https://groups.google.com/forum/#!forum/mongodb-user>`_.
#. Write a blog post about how you use MongoDB.
#. Attend (and present at) `MUGs <http://www.mongodb.com/user-groups>`_ or `other conferences <http://www.mongodb.com/talk-proposal>`_.
#. Contribute code to the MongoDB server.
#. Contribute code to one of the MongoDB, Inc.-supported Drivers.
#. Contribute to one of the `Community Supported Drivers <http://docs.mongodb.org/ecosystem/drivers/community-supported-drivers/>`_ (Not managed by MongoDB, Inc.).
#. Contribute to the `MongoDB Manual <http://docs.mongodb.org/manual/about/>`_.

This section focuses on code contribution, namely, to help you get
started with a MongoDB project as well as to get your code approved and
merged into the master branch.
